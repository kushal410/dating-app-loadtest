# Dating App Load Testing

This is a GitHub Actions setup to perform API load testing
for your Flutter + Node.js dating app using k6.

## Setup

1. Update `loadtest/loadtest.js` with your API endpoints.
2. Adjust `vus` (virtual users) and `duration` as needed.
3. Commit and push to GitHub.

## Run Test

1. Go to GitHub Actions.
2. Select **Dating App Load Test** workflow.
3. Click **Run workflow**.
4. View logs for performance metrics.
